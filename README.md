# 2020-2 마이컴 프로그래밍 응용 과제

### 목표
- 적외선 센서를 이용한 라인 트레이싱
  - 양쪽 모두 검은 경우: 장애물 감지
  - 그 외: 검은 선 따라 전진
- 초음파 센서를 이용한 장애물 감지
  - 장애물이 없는 경우: 전진
  - 한족에 장애물이 있는 경우: 장애물이 있는 쪽으로 90° 회전
  - 양쪽에 장애물이 있는 경우: 반바퀴 회전 후 전진(반대 방향으로)
